# Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории. MacOS, Linux
```sh
pwd
```

Листинг текущей диретории 
Windows:
```
dir
```
Linux, MacOs
```sh
ls
```

Удаление файла
```sh
del <filename>
```
в Linux, MacOs
```sh
rm <filename>
